**THE BLOCK GAME**

This is the block game. This game consists of a boards of blocks. Each block is either a solid color block
or it is divided into four smaller blocks. You can play alone or against friends up to 4. You get to choose
how many turns each player gets. The points are counted after all turns are used.

There are two different goals one of them consists of getting the most blocks from a certain color to touch the
perimeter of the square. And the other one consists of forming the largest blob of the given color. 

To achieve this you have 3 possible moves. You can **rotate** the block clockwise or counterclockwise with the
keys 'R' and 'L'. You can **swap** four sub-blocks vertically or horizontally with keys 'V' and 'H'. And finally
**smash** the selected block to create 4 new random blocks using 'S'. You cannot smash the top level block or a
block at the lowest depth.

To play this game you need to fist download pygames. To do this run "python3 -m pip install -U pygame" in your terminal
(without the quotations).

This game utilizes trees and recursion as well as other basic concepts in python.
